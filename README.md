# contact-list-app
A simple contact list application built using node.js, express.

By doing this simple project i learnt about the following things:
  * How to do basic CRUD operations using Mongoose
  * How to access Mongo DB hosted on MongoLab
  * Two-Way data binding in Angular.js
  
# How to run the project:
  Just clone the repo.
  Go to the folder where you unzipped the folder, using the terminal
  
  Run the following code to install the dependencies
  `npm install`
  and then run  `node server.js` to run the server.
  
  Now go to `http://localhost:3000` to see the application.
